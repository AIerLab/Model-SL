# Model-SL

1. **Clone the repository**:
   
   Clone the main repository along with its subrepositories using:
   ```bash
   git clone --recurse-submodules https://github.com/AIerLab/model-sl.git
   ```


2. **Cloud Execution**:

     ```bash
     bash run_cloud.sh
     ```

3. **Local Execution**:
     
     ```bash
     bash run_local.sh
     ```
